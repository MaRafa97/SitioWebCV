# Rafa Cruz — CV Website

This is a simple, responsive CV-style website built for a software engineer. Open `src/index.html` in your browser to view the site.

How to run locally:

1. Open the project folder in your file explorer.
2. Double-click `src/index.html` to open in your default browser, or run a simple static server:

```powershell
# from repository root
cd src; python -m http.server 8000; Start-Process http://localhost:8000
```

Files added:
- `src/index.html` — main page
- `src/css/styles.css` — styles
- `src/js/script.js` — small JS interactions
- `src/assets/images/` — avatar and company badges

Edit the email, locations, or content as needed for accuracy.
